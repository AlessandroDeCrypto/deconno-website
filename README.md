# De Conno Limousinen Service Website Starter

This is a complete Next.js starter project for De Conno Limousinen Service.

## Features
- English-only copy
- Premium homepage
- Services page
- Fleet page
- About page
- Contact page
- Tailwind styling
- SEO metadata for Zurich Airport / Kloten / Zurich City / Switzerland transfers

## Local setup

```bash
npm install
npm run dev
```

Open:

```txt
http://localhost:3000
```

## Before going live
1. Replace image placeholders with real photos or AI images.
2. Connect the contact form to email or WhatsApp.
3. Add route-specific landing pages.
4. Deploy to Vercel.
